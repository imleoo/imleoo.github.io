## Customization

### Colors
Edit the CSS variables in `assets/css/main.css`:
```css
:root {
  --primary-color: #2563eb;
  --secondary-color: #7c3aed;
  --accent-color: #06b6d4;
  /* ... more variables */
}
```

### Content
Update the main content in `index.md` and individual page files.

### Navigation
Modify the navigation menu in `_config.yml`:
```yaml
navigation:
  main:
    - title: "üè† Home"
      url: /
    # ... more items
```
